# 46009361.com

This is my personal website. To save on AWS costs, I've moved to GitHub Pages, as dynamic site generation was unnecessary for this site.

## Features

- Links to some of my other projects
- Old URL paths (e.g. `/vote/index.php`) still work
- Dark theme, if your browser is set to that

## Running locally

This is for `http://localhost:8000/`. You may add a port number to the end of the last command to change the URL.

```zsh
git clone https://github.com/46009361/46009361.com.git
cd 46009361.com
python3 -m http.server
```
